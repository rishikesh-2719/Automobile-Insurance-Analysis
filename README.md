# Automobile Insurance Analysis

## Project Overview

This project analyzes automobile insurance data to evaluate insurance company performance, customer complaints, and complaint ratios. The analysis helps identify trends and provides business insights for improving customer satisfaction and operational efficiency.

The project was developed using Python, Pandas, and data visualization techniques.

---

## Objectives

* Analyze automobile insurance company performance.
* Evaluate customer complaint trends.
* Compare complaint ratios across insurers.
* Identify companies with the highest and lowest complaint rates.
* Build dashboards for business reporting.

---

## Technologies Used

* Python
* Pandas
* Jupyter Notebook
* Excel
* Matplotlib
* Seaborn
